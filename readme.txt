Mini Juego

El objetivo es adivinar el número que podrá estar entre 1 y 20.
Comienza con score de 10 puntos.
Cada intento de acierto podrá ser ganador (el número ingresado coincide con el número secreto) ó fallido restnado un punto del score. En este último caso el juego te indica si es mayor o menor para realizar otro intento.
Si el puntaje llega a 0 Game Over.
Mediante "Nueva Partida" podemos volver a jugar.
Además, el juego muestra el HighScore que obtuvimos entre todas las partidas jugadas.


Mini juego basado en Workshop de Nerdearla 2021.
Utilizando HTML, CSS y JS.
Versión propia:
Utilizo Boostrap.
Utilizo form en lugar de table para elegir número y Probar. Evito que se actualice en JS.
El valor se indica como required. Se mantiene el condicional en JS ante el caso de "no ingresa valor" igualmente.
Cuando hay match y gana se modifica backgroundColor de una section, no del body.
Cuando hay GameOverse modifica backgroundColor de una section y muestra el Número Secreto.
Score inicial 10 puntos en lugar de los 20 de la versión original del juego.
